# Анализ продаж розничного магазина (SQL + Excel)

## Цель проекта
Понять, какие товары приносят основную выручку, как скидки влияют на прибыль, есть ли сезонность.

## Данные
Использован открытый датасет Superstore Sales (Kaggle). Содержит 9994 заказов за 2014-2017 годы.

## Инструменты
- SQLite (язык запросов)
- Excel (сводные таблицы, дашборд)
- Python (pandas) для первичной очистки

## Ключевые выводы
1. Категория *Technology* приносит 36% выручки, но *Office Supplies* — самую высокую маржинальность.
2. Скидка выше 30% делает продукт убыточным в 80% случаев.
3. Пик продаж — ноябрь и декабрь (сезон подарков).

## Как посмотреть результаты
- Файл `queries.sql` содержит все запросы
- В папке `results/` лежат выгрузки
- Дашборд в Excel: `dashboard/superstore_dashboard.xlsx`
